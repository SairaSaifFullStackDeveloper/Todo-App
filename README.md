# Getting Started with Todo App

This project is created in MERN Stack Package(MongoDB, Express.js, React.js and Node.js) on VS Code.


## Available Scripts

In the project directory, you can run:

Firstly, Go to frontend folder and run the command:

### `npm install`

After this run the below command:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

After this go to Server folder and run the command:

### `npm install`

Now change the database connection link and run the command:

### `npm start`

Now app successfully run and if you face any issue the contact me <a href="https://github.com/SairaSaifFullStackDeveloper/Todo-App/issues">Here</a>
